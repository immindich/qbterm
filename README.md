# qbterm

A terminal-based interface for qBittorrent.

Built with [Ink](https://github.com/vadimdemedes/ink) and TypeScript.

## Features

- View and sort torrents
- Add new torrents (currently only supports adding via URL)
- Pause and resume
- Reconnect after network disruption
- Automatically log in again after session expires or if qBittorrent restarts
